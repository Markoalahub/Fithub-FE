# 데모 로그인 계획

## 1. 목업 계정 정보
| 역할 | 이름 | 아이디(이메일) | 비밀번호 |
| --- | --- | --- | --- |
| 기획자(PM) | 김기획 | `pm@fithub.demo` | `pm1234!` |
| 개발자(DEV) | 이개발 | `dev@fithub.demo` | `dev1234!` |

## 2. 역할별 로그인 후 이동 계획
- 기획자(PM) 로그인
  - 기본 진입: `기획자 대시보드`
  - 접근 가능: `기획자 대시보드`, `관리자 설정`
- 개발자(DEV) 로그인
  - 기본 진입: `개발자 대시보드`
  - 접근 가능: `개발자 대시보드`만

## 3. 로그인 방식
- 소셜 로그인 선택: `GitHub` 또는 `카카오`
- 현재는 API 미연동 데모 모드이므로, 위 목업 계정으로만 인증 처리
