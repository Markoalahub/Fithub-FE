# Fithub FE

React + TypeScript + Tailwind(v4) 기반 Fithub 프론트엔드 프로젝트입니다.

## Run Locally

**Prerequisites:** Node.js 20+

1. Install dependencies:
   `npm install`
2. Create `.env` from `.env.example` and set values
3. Run the app:
   `npm run dev`
